# CRUX - B2B Construction Materials Marketplace

## Overview
CRUX is a B2B marketplace connecting contractors with suppliers through group buying and cross-docking logistics.

## Architecture
- **Backend**: Node.js + TypeScript + Prisma + PostgreSQL
- **Frontend**: Flutter
- **Deployment**: Docker Compose

## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
flutter pub get
flutter run
```

## Documentation
See `/docs` for detailed documentation.

## Project Structure
- `/backend` - REST API and business logic
- `/frontend` - Flutter mobile app
- `/docs` - Documentation
- `/.agent` - AI agent configuration
